# Global Templates

Artifact templates for target projects. Copy or generate from these when creating REQ, TASK, BUG, TC, validation reports and ADRs.

| Template | Use |
|----------|-----|
| `requirement.template.yaml` | Automation / validation export for REQ |
| `task.template.yaml` | TASK-XXXX in TASKS.md or PLAN.md |
| `bug.template.yaml` | BUG-XXXX in Test/{{AREA}}/bugs/ |
| `test-case.template.yaml` | TC-XXXX in Test/{{AREA}}/test-cases/ |
| `validation-report.template.json` | Post-implementation quality report |
| `architecture-decision-record.template.md` | DEC-XXXX |
| `milestone-catalog.md` | Suggested milestones for #project-setup |

Primary REQ format on target projects is **Markdown** in `Design/areas/{{AREA}}/REQUIREMENTS.md`.
