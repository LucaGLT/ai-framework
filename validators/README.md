# Validators

Quality scoring and schema validation for framework artifacts.

| File | Purpose |
|------|---------|
| `validation-rubric.yaml` | Weights, thresholds, blocking conditions |
| `validation-report.schema.json` | JSON schema for validation reports |

See `templates/validation-report.template.json` for report structure.

Gate default: **disabled** (`VALIDATION_GATE_ENABLED=false`) until post-pilot.
